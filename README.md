# 📊 Sales Analysis Dashboard – Power BI Report

## 📌 Project Overview
This repository contains an interactive **Sales Analysis Dashboard** developed using **Power BI**,to analyze and visualize sales performance data effectively.
The project focuses on analyzing customer purchasing behavior, product performance, and regional sales trends through visually appealing and interactive dashboards.

The dashboard helps businesses understand sales patterns and make data-driven decisions using powerful visualizations and insights.

The report contains two interactive tabs, each focused on different aspects of the sales analysis.

---

# 🎯 Feature
- Analyze overall sales performance
- Interactive Power BI Dashboard
- Dynamic Filters & Slicers
- Sales and Order Analysis 
- State Sales and order Analysis
- Product Performance Analysis
- Regional Sales Insights
- Gender & Age Group Analysis
- Occupation based Sales Analysis

---

# 📑 Dashboard Structure

## 1️⃣ Sales Dashboard
This dashboard provides detailed sales insights, including:

### 📌 Visualizations Included
- Product wise Sales Analysis
- Top Ordered Product
- Product Orders Analysis
- Sales Comparison between Married & Unmarried Customers
- Sales Sales Comparison between Female & Male Customers
- Sales by Occupation
- Sales by Age Group
- Top 5 Product Sales According to Gender

The dashboard allows users to interact with charts and filters for deeper analysis and better understanding of customer behavior and sales performance.
